Calculadora 
Este repositório contém uma calculadora simples desenvolvida em Python, capaz de realizar as quatro operações aritméticas básicas: adição, subtração, multiplicação e divisão. É um projeto direto, focado na funcionalidade essencial e na clareza do código.

Funcionalidades

*   **Adição:** Soma dois números.
*   **Subtração:** Subtrai o segundo número do primeiro.
*   **Multiplicação:** Multiplica dois números.
*   **Divisão:** Divide o primeiro número pelo segundo, com tratamento para divisão por zero.
*   **Interface de Linha de Comando:** Menu interativo para seleção de operações.
*   **Validação de Entrada:** Garante que apenas números válidos sejam processados.

Como Executar

Siga os passos abaixo para baixar e executar a calculadora em sua máquina.

Pré-requisitos

*   **Python 3.x:** A calculadora é compatível com Python 3. Certifique-se de ter o Python instalado. Você pode verificar sua versão com `python --version` ou `python3 --version`.
